### 프로젝트 목적: 
자동차의 손상된 부위 (스크레치/찌그러짐)에 대한 이미지를 학습한 모델을 활용하여, 추후 자동차 손상된 부위를 분류하고 해당 부위의 넓이를 도출하여 단위 면적당 자동차 수리비용을 예측하는 프로젝트 진행.
